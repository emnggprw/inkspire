   ```sh
   git clone https://github.com/emnggprw/inkspire.git
   cd inkspire
   ```
2. Install dependencies:
   ```sh
   flutter pub get
   ```
3. Create a `.env` file and add your OpenAI API key:
   ```sh

   ```

